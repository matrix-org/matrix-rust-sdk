The thread cache, aka `ThreadEventCache`, has been extracted from the room
cache, aka `RoomEventCache`. To get a thread cache, please use the
`EventCache::thread()` method.

The `RoomEventCache::thread_pagination` and `subscribe_to_thread` methods have
been removed.

Before:

```rust
let room_cache = event_cache.for_room(room_id).await?;
let thread_pagination = room_cache.thread_pagination(thread_id).await?;
let thread_subscriber = room_cache.subscribe_to_thread(thread_id)?;
```

After:

```rust
let thread_cache = event_cache.thread(room_id, thread_id).await?;
let thread_pagination = thread_cache.pagination(); // no async, no error!
let thread_subscriber = thread_cache.subscribe().await?;
```

Also, the `EventCache::for_room` method has been renamed `room`.
