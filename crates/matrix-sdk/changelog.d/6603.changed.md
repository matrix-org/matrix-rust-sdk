The event-focused cache, aka `EventFocusedCache`, has been extracted from the
room cache, aka `RoomEventCache`. To get an event-focused cache, please use the
`EventCache::event_focused()` method.

The `RoomEventCache::get_or_create_event_focused_cache` and
`get_event_focused_cache` methods have been removed.

Before:

```rust
let room_cache = event_cache.for_room(room_id).await?;
let event_focused_cache = room_cache.get_or_create_event_focused_cache(thread_mode, number_of_initial_events).await?;
```

After:

```rust
let event_focused_cache = event_cache.event_focused(room_id, event_id, thread_mode, number_of_initial_events).await?;
```
